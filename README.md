# vlc-handimote
A free and open-source app to control your VLC Player.

## Requirements
For the app to talk to VLC on your computer, make sure:

- VLC is running on your PC.

- Web Interface is enabled in VLC (Preferences > All > Interface > Main interfaces > Check "Web").

- A password is set in the VLC Lua HTTP settings.

- Both devices are on the same Wi-Fi.
